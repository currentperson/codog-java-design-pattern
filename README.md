# 前言

本书主要介绍自己对设计模式的一些学习, 我觉得设计模式不能脱离生产来谈, 所以整本书我们会把自己当做一个银行的开发人员, 讲某个模式的时候都会尽量找到对应的业务场景

本书内容可转载, 但是希望注明出处
